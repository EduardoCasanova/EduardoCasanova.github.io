Bucles

En general se encuentran tres formatos distintos cuando se debe repetir una serie de acciones:

for

Se usa cuando se conoce de antemano la cantidad de repeticiones a realizar.

Formato

for variable:= valor_inicial to valor_final do

Ejemplo: Para cada mes del año el usuario ingresa la cantidad de lluvia caida. Se quiere conocer la precipitación total anual y el promedio mensual.

Resolución: En este caso se debe prestar atención a las palabras para cada mes del año, que indican que se necesita repetir la lectura 12 veces.

¿Qué variables necesito?

Cada dato ingresado se procesa directamente, por lo que una sola variable sirve para los 12.

Se piden dos valores de salida (suma y promedio), pero el promedio se puede calcular directamente dentro del write.

suma es una variable acumuladora por lo que se debe inicializar en cero.

la variable entrada recibe datos desde el teclado, por lo que no es necesario inicializarla.

No olvidarse que el for necesita su variable contadora, en este caso entera.

Ver código

program for1;
var 
    suma, entrada, i : integer;
    
BEGIN
    {inicialización de variables}
        suma := 0;
        writeln('ingrese el valor mensual (solo números enteros)');
        for i := 1 to 12 do begin
            write ('mes : ', i, ' --> ');
            readln (entrada);
            suma += entrada;    {es una forma resumida de suma = suma + entrada}
        end;
            writeln ('precipitación anual : ', suma);
            writeln ('promedio mensual : ', (suma / 12):2:1);
            {:5:2 le indican a writeln que debe imprimir con 5 caracteres de ancho mínimo y 1 decimal}
END.
