Programación

Aquí encontrarán algunas notas y tutoriales que creo principalmente como documentación propia, pero que pueden ser útiles para otras personas.

Para comenzar, publico mis apuntes sobre la creación de sitios web integrando Eleventy (Javascript + HTML + Markdown + Nunjuks + ...), TailwindCSS (PostCSS), Snowpack, y algunas librerías javascript específicas como ser MathJax para el contenido matemático.

Nota: Programo sobre GNU/Linux, pero en general lo que sigue sirve para otros sistemas operativos (a lo sumo tendrán que buscar la instrucción correspondiente si algo no funciona en la consola y reemplazar algunos '/' por '\' en las rutas de los archivos).
En general sobre Node uso scripts y paquetes sin problemas de compatibilidad entre plataformas.
